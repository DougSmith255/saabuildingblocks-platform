# Canva Image Generation Style Guide

> Reference file for AI-generated images via Canva MCP.
> All generated images go to the **Web Photos** folder in Canva (ID: FAHCGUjeqbs).

## Visual Theme

- **Aesthetic:** Futuristic, space, cyber/tech
- **Background:** Deep black (#0a0a0a) or dark space scenes with subtle nebula/star elements
- **Primary accent:** Gold/yellow (#ffd700)
- **Secondary accent:** Electric green (#00ff88) — used sparingly
- **Text color:** Off-white (#e5e4dd)
- **Mood:** Professional, aspirational, clean, high-tech
- **Effects:** Subtle glows, light rays, lens flares — never overdone

## Photography Direction

- **Subject matter:** Real estate professionals, modern offices, technology, cityscapes, luxury homes
- **Demographics:** Reflect actual US demographic proportions naturally — no forced or artificial diversity quotas
- **Tone:** Confident, professional, successful — agents who are thriving
- **Avoid:** Stock photo clichés, overly staged poses, cheesy thumbs-up, clipart aesthetics

## Design Rules

- Clean, modern sans-serif typography
- Generous whitespace / negative space
- Gold (#ffd700) for highlights, labels, key data points
- Dark card/panel UI elements with subtle gold borders
- No bright/white backgrounds — always dark theme
- Minimal text on photos — let imagery speak
- SAA brand watermark when appropriate: "Smart Agent Alliance — smartagentalliance.com"

## Canva Settings

- **Folder:** Web Photos (FAHCGUjeqbs)
- **Brand kit:** None (custom style per above, not tied to eXp brand kit)
- **Preferred design types for web images:** poster, facebook_post, instagram_post, your_story
- **For infographic-style content:** Use presentation (single slide, 16:9 landscape)

## Prompt Template

When generating images, always include these elements in the query:

```
Futuristic dark space theme. Black background (#0a0a0a). Gold/yellow (#ffd700) accents.
Clean modern typography. Cyber/tech aesthetic with subtle glow effects.
Professional real estate context. Demographics reflecting natural US population proportions.
[SPECIFIC CONTENT DESCRIPTION HERE]
```
